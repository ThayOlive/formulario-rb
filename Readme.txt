para rodar o app na sua máquina :

1. Clone o repositório 
2. Acesse a pasta com "Abrir com Code"
3. crie um ambiente virtual no terminal do VScode:
no cmd windows digite: 
    python -m venv venv
ative o ambiente virtual: 
    digite: venv\scripts\activate
instale os requisitos:
    digite: pip install -r requirements.txt
Abrir no navegador:
    digite: manage.py runserver
